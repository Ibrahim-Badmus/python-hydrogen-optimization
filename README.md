# Hydrogen Process & Systems Engineering Portfolio

## Overview
This repository contains engineering models, simulations, and analyses related to
hydrogen and ammonia systems, developed using industry-standard tools.

## Scope
- Process modeling (Aspen Plus / HYSYS)
- Python-based numerical analysis and optimization
- Techno-economic evaluation of hydrogen and ammonia systems

## Tools Used
- Aspen Plus
- Python (NumPy, SciPy, Pandas, Matplotlib)
- Microsoft Excel

## Contents
- Models
- Scripts
- Documentation
- Results and figures

## Purpose
This work is part of a professional development program focused on mastering
hydrogen and ammonia process systems and translating theory into real-world value.
